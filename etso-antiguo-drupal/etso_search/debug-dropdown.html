Debug: Dropdown de Textos - ETSO Search

Instrucciones

Abre la consola del navegador (F12) y ejecuta estos comandos en la página del buscador:

1. Verificar si existen los elementos

console.log('Botones textos-toggle:', document.querySelectorAll('.textos-toggle')); console.log('Wrappers:', document.querySelectorAll('.textos-dropdown-wrapper')); console.log('Dropdowns:', document.querySelectorAll('.textos-dropdown'));

2. Verificar estructura HTML de un dropdown

var wrapper = document.querySelector('.textos-dropdown-wrapper'); if (wrapper) { console.log('Wrapper HTML:', wrapper.outerHTML); console.log('Botón:', wrapper.querySelector('.textos-toggle')); console.log('Dropdown:', wrapper.querySelector('.textos-dropdown')); } else { console.log('No se encontró wrapper'); }

3. Verificar si el behavior está cargado

console.log('Drupal behaviors:', Drupal.behaviors); console.log('etsoSearchBuscador:', Drupal.behaviors.etsoSearchBuscador);

4. Probar manualmente el toggle

var button = document.querySelector('.textos-toggle'); if (button) { var wrapper = button.closest('.textos-dropdown-wrapper'); var dropdown = wrapper.querySelector('.textos-dropdown'); console.log('Dropdown hidden:', dropdown.hasAttribute('hidden')); // Toggle manualmente if (dropdown.hasAttribute('hidden')) { dropdown.removeAttribute('hidden'); console.log('Dropdown mostrado'); } else { dropdown.setAttribute('hidden', ''); console.log('Dropdown ocultado'); } } else { console.log('No se encontró botón'); }

5. Verificar event listeners

var button = document.querySelector('.textos-toggle'); if (button) { console.log('Event listeners en el botón (no se pueden listar directamente)'); console.log('Probar click manualmente:', button.click()); } else { console.log('No se encontró botón'); }

6. Re-ejecutar el behavior manualmente

if (typeof Drupal !== 'undefined' && Drupal.behaviors.etsoSearchBuscador) { Drupal.behaviors.etsoSearchBuscador.attach(document, drupalSettings); console.log('Behavior re-ejecutado'); } else { console.log('Behavior no disponible'); }

7. Verificar errores en consola

Mira la consola para ver si hay errores JavaScript. Los errores comunes incluyen:

  • Cannot read property 'querySelector' of null - El wrapper no existe
  • Cannot read property 'textContent' of null - El .btn-arrow no existe
  • once is not defined - La librería once no está cargada

Posibles problemas y soluciones

  • El dropdown no se muestra: Verifica que el CSS no tenga display: none !important
  • El behavior no se ejecuta: Verifica que la librería esté cargada en etso_search.libraries.yml
  • Los elementos no existen: Verifica que el template Twig esté generando el HTML correctamente
  • Click no funciona: Verifica que no haya propagación de eventos desde la fila de la tabla
